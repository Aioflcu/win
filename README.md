# FemiCoin (Demo)

A demo React + Vite + Tailwind crypto dashboard called "FemiCoin".

Features:
- Login page (hardcoded credentials: `admin` / `1234`)
- Dashboard with header, wallet overview, top cryptos (CoinGecko), quick actions, transactions, news, footer
- Withdraw modal that marks transactions as Pending and updates local state
- Responsive layout using Tailwind CSS

Quick start

1. Install dependencies

```bash
cd /Users/oyelade/WICN/femecoin
npm install
```

2. Start dev server

```bash
npm run dev
```

Open http://localhost:5173 and login with `admin` / `1234`.

Notes
- This is a demo. No real funds are transferred.
- Coin data is fetched from the public CoinGecko API (rate limits may apply).
